# Customer Personality Analysis - Data Cleaning

## Objective
Clean and preprocess a customer personality dataset to make it ready for analysis.

## Tools Used
- Python
- Pandas
- Excel

## Tasks Performed
- Handled missing values
- Removed duplicate records
- Standardized data formats
- Converted date columns
- Cleaned column names
- Verified data types

## Dataset
- 5,000 customer records
- 21 features

## Outcome
Created a clean and structured dataset suitable for data analysis and visualization.
